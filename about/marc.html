Welcome!

This is a demo of the Infrared "search framework", a Blacklight lookalike. It illustrates what the Blacklight MARC demo Solr collection looks like in this different framework.

Read the Blacklight documentation. Read the Infrared documentation.

About this instance

This demo uses the Solr core (collection) used in the Blacklight demo — the sample set of MARC bibliographic records that ships with Blacklight — served through Infrared Python Flask app rather than the Blacklight Ruby on Rails application. The Solr core is the one that is provisioned using Blacklight's own schema (the same Solr dynamic field conventions Blacklight uses: _tsim, _ssim, _si, and friends), so the facets, sort options, and display fields you see here mirror the Blacklight demo's catalog_controller.rb.

It exists to exercise Infrared against a real-world MARC/Blacklight index:

  1. Search across the corpus, or scope a search to a single field.
  2. Narrow results with the facets on the left (format, author, publication year, topic, language, call number, region, era).
  3. Sort by relevance, year, author, or title, and switch between the list, table, and gallery display modes.
