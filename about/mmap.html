How it works

What we have done is combined records from the various tables together using a common key, to wit, the Artifact number.

A bit of pipeline plumbing does the merging and the results are fed into Solr. Solr presents the data using a basic keyword search box (at the top) and, more marvelously, arranging many of the fields in "facets" at the left. The user can use these in combination to generate a search result containing the desired information.
