The TAP Finder

This website provides a means to "slice-and-dice" some of the myriad data elements of the TAP project.

The diagram below illustrates how 6 distinct but related data sets are combined to create a single interface that presents images (photos, scans, etc.) and metadata in an easy-to-use manner. The interface consists of a basic keyword search box (at the top) and, more marvelously, arranging many of the fields in clickable "facets" at the left. The user can use these in combination to generate a search result containing the desired information.

Figure 1. Data flow and components of the TAP Finder

This search engine allows you to search for 6 types of TAP data:

  • Master bag log records (from Excel Spreadsheet)
  • Photolog records (from Excel Spreadsheets)
  • Bags (from Excel Spreadsheets)
  • Images (from Box and Prof. Pigott's various hard drives and USB sticks)
  • Inventory records (from Prof. Voelkers's Penn Museum inventory)
  • Files in the shared Box folder

... and combinations of the above

How it works

This is a work in progress! Caveat utilizator!

What we have done is combined the various records together where possible using a common key. There are two keys in use:

T# key
When a record (i.e. a spreadsheet row or image filename) contains a T#, all the data associated with the T# can be listed together. So for numbered artifacts, all the images, bag log info, photolog info, etc. can be consolidated.
Image key
The filenames of the images do not usually carry much information. Typically, the filename of an "official" image includes the Season, the Roll # and the Exposure number, but occasionally additional information is included in the filename e.g. Tap 94 Rol 168 #31 opc T#30691.tif. Using this information, we can match the images to the corresponding photolog fields, which provides a more complete description of the images.

The search engine

Once the data is combined into a coherent whole, it can be served to users via a web browser. The merged records are loaded into Solr — a blazingly fast, open source search engine — which provides the keyword search and faceted browsing you see here.
